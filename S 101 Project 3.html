Answer the following questions on your page:



What type of file is your web page saved as?

A file with “.HTML” at the end.

What are the three major tags the most basic web page needs?

  • ”body and /body”
  • ”h# and /h#”
  • ”p and /p”

Where does the Head of a web page display when viewed in a web browser?

Where the top of the webpage is.

Where does the Body of a web page display when viewed in a web browser?

Where the bulk of the webpage is, below the top.
